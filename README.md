#alfred 2 进制转换插件
> 命令 **cov**
#简介
> 支持自动判断输入字符并转换成其他进制，目前支持16进制，10进制，8进制，2进制之间的任意转换
输入格式：
16进制：以0x开头
<code>
cov 0xFFFF
</code>
8进制：以0c开头
<code>
cov 0x7777
</code>
2进制：以b开头
<code>
cov b1111
</code>
10进制：无前缀，直接输入数字
<code>
cov 9999
</code>
回车即复制到剪贴板

#例子
##16进制转换为其他进制
<code>
cov 0xFFFF
</code>
结果：
2进制：b1111111111111111
8进制：0c177777
10进制：65535
